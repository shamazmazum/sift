API

gaussian-scale-space(a &key (σ 1.6d0) (n 3) (m 3))
Compute a set of blurred images of A which constitutes a scale space of A with M octaves, N images per octave. Each octave also contains one image below it and two images above it.
keypoints(scale-space)
Detect keypoints in a scale space.
descriptors(scale-space)
Calculate keypoints and descriptors (128-element vectors) for a scale space.
describe-point(keypoint scale-space)
Describe a keypoint in a scale space. The descriptor is a vector of 128 float values which is invariant to linear transformations of the image which produced this scale space.
find-matches(set1 set2)
Find matching keypoints in SET1 and SET2. A point from the first set can have at most one corresponding point from the second set. Algoritmic complexity: \(O(n \cdot m\)), \(n\) and \(m\) being numbers of elements in each set.
